# PDF Merger App - Installation Guide

## For Other Computers

### Simple Installation (Recommended)

1. **Copy the executable file:**
   - Take the file: `dist/PDF Merger.exe`
   - Copy it to the other computer (USB drive, email, cloud storage, etc.)

2. **Run the application:**
   - Simply double-click `PDF Merger.exe`
   - No installation needed - it's a standalone application
   - Windows may ask for permission to run - click "Run anyway"

3. **First Use:**
   - When you first run the app, it creates folders in:
     - `C:\Users\YourUsername\AppData\Local\PDF Merger App\input` (for source PDFs)
     - `C:\Users\YourUsername\AppData\Local\PDF Merger App\output` (for merged PDFs)

### Using the Application

1. **Select Letterhead PDF:**
   - Click "Browse" to select a letterhead PDF from anywhere on your computer
   - Or use "From Input Folder" to select from pre-placed files

2. **Select Content PDF:**
   - Click "Browse" to select a content PDF
   - Or use "From Input Folder"

3. **Merge:**
   - Click "🔀 Merge PDFs"
   - The merged PDF will be saved to the output folder

4. **Access Folders:**
   - Click "📂 Open Input Folder" to add files
   - Click "📂 Open Output Folder" to get your merged PDFs

### Troubleshooting

**Q: "Failed to execute script" error?**
- A: Make sure you have the latest version of the exe from the dist folder
- Try running it as Administrator: Right-click → Run as administrator

**Q: Where are my output files?**
- A: Check the output folder shown in the application window
- Typical location: `C:\Users\YourUsername\AppData\Local\PDF Merger App\output`
